Positioner.js

jQuery random sizing and positioning.
It's like the anti-masonry.

Apply $.positioner to the container element.
Positioner requires a minimum of the following two parameters:

  1. item: the child elements selector to be positioned
  2. widths: array of widths to apply to the positioned elements
$(".container").positioner({
	item: ".target",
	widths: [40, 80, 100]
});
